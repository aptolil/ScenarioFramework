This is a small framework I have developed with the intention to make it simpler for any one to make their own scenarios in Simrail. (https://wiki.simrail.eu/).

For a description of how touse the framework se the scenario readme file in `Scenarios/Scenario_1/ReadMe/en/README.md`.

The scenario readme files can be viewed within Simrail but code examples seams to be formated incorrectly so it is better to open the file directly in an editor.

[PL]

To niewielki framework, który opracowałem z myślą o ułatwieniu każdemu tworzenia własnych scenariuszy w Simrail. (https://wiki.simrail.eu/).

Opis korzystania z frameworka można znaleźć w pliku readme scenariuszy w `Scenarios/Scenario_1/ReadMe/en/README.md`.

Pliki readme scenariuszy można przeglądać w Simrail, ale przykłady kodu wydają się być niepoprawnie sformatowane, dlatego lepiej otworzyć plik bezpośrednio w edytorze.
